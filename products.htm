Test Kits

All of VMRD’s ELISA kits are USDA licensed and offer diagnostic tools for a variety of species including bovine, equine, caprine, and ovine.

Reagents

VMRD reagents include slides, monoclonal antibodies, conjugates, and more. These products target both large and companion animal diseases.

ELISAPro

VMRD is pleased to be able to offer an automation solution for enhanced workflow. Our ELISA processor is able to automatically run and provide results for all of VMRD’s assays.

Serology Reference Testing

VMRD provides reference testing for select FA and ELISA applications. Samples should be shipped to the attention of "VMRD Serology Reference Testing" and all samples must have a sample submission form indicating the tests requested
